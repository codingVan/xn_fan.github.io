语言模型学习

语言模型 (Language Model) 用来判断一个语句是否符合语义。在实际应用中，常常利用概率来解决这一问题。

给定一个句子 $s$，由 $n$ 个单词 $w_1, w_2, w_3,…,w_n$ 组成，为了判断这个句子是否通顺，可以利用
$p(s) = p(w_1, w_2, w_3,…,w_n)$ 来表示，即这个单词序列出现的概率。

如：$p( It\ is\ good) > p(good\ is\ it)$ 可以认为前面一句话在语法上更为合理。

Markov Assumption

在应用中，我们常常不会直接用联合概率分布来表示语句，而是利用 Markov Assumption 来进行计算。如一阶 Markov Assumption 情况下，$p(w1, w_2, w_3,…,w_n) = p(w_1)p(w_2|w_1)p(w_3|w_2)…p(w_i|w{i-1}) = p(w1)\prod{i=2}^np(wi|w{i-1})$。

二阶，三阶或是更高阶同理。

N-Gram

Unigram

$p(w_1, w_2, w_3,…,w_n) = p(w_1)p(w_2)p(w_3)…p(w_n)$

即单词间不存在依赖性，这种情况不考虑语序，出现了同样单词的两句话即使顺序差别很大，也会得到相同结果，不是很合理。

Bigram

$p(w1, w_2, w_3,…,w_n) = p(w_1)p(w_2|w_1)p(w_3|w_2)…p(w_i|w{i-1}) = p(w1)\prod{i=2}^np(wi|w{i-1})$。

即一阶马尔科夫假设的情况，当前词只与上一单词有关。

同理可以推广至N-Gram。

如何评价？

对于一个语言模型，我们引入疑义度 (Perplexity) 对其进行评价。

$perplexity = 2^{-x}$

这里 $x: average\ log\ likelihood$，$x$ 越大，说明模型使得语料库中单词平均出现的概率增大，拟合效果好，此时疑义度越小。作为一种无监督的评价手段，现在疑义度也常常作为深度学习衡量模型能力的一种指标。

平滑

如果语句没有在语料库中出现，其概率为0，此时引入平滑项进行缓解。

Add-One Smoothing

$p(wi|w{i-1})old = \frac{c(w_i-1,w_i)}{c(w_i-1)}$，这里 $c(w_i)$ 表示该词在语料中出现的次数。

$p(w_i|w{i-1})_{Add-1} = \frac{c(w_i-1,w_i)+1}{c(w_i-1)+|V|}$，$|V|$代表词库大小，引入使得概率和为1。

Add-K Smoothing

$p(wi|w{i-1})_{Add-K} = \frac{c(w_i-1,w_i)+K}{c(w_i-1)+K|V|}$

Interpolation Smoothing

计算 $Trigram$ 时同时考虑 $Unigram$ 和 $Bigram$


$p(wi|w{i-1}, w{i-2}) = \lambda_1p(w_i|w{i-1}, w{i-2})+\lambda_2p(w_i|w{i-1})+\lambda_3p(w_i)$

$\lambda_1+\lambda_2+\lambda_3 = 1$

Good-Turning Smoothing

$Nc$ 代表语料中出现 $c$ 次的单词个数
$p{GT} = \frac{(c+1)N_{c+1}}{N_cN}$，考虑了没有出现过的单词的概率，需要引入平滑解决 $N_c$ 比较稀疏时的情况。


语言模型同样可以作为一种生成模型，用来生成新的语句。
